CSS Positioning Demonstration

Paragraph 1 (Relative Position): This paragraph has position set to relative. Relative positioning allows an element to be positioned relative to its normal position in the document flow. The element still takes up space in the normal layout.

Paragraph 2 (Float): This paragraph has float property set to left. Floating an element removes it from the normal document flow and allows other content to wrap around it. This is commonly used for laying out multiple elements side by side.

Paragraph 3 (Absolute Position): This paragraph has position set to absolute. Absolute positioning removes the element from the normal document flow and positions it relative to its nearest positioned ancestor (or the viewport). The element does not take up space in the normal layout.

This paragraph is here to demonstrate how floating and absolute positioning work with other content.
